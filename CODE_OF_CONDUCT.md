# Code of Conduct

## Our Commitment
We are committed to making participation in this project a respectful and harassment-free experience for everyone.

## Expected Behavior
- Be respectful and constructive.
- Focus on technical issues, not individuals.
- Accept feedback professionally.

## Unacceptable Behavior
- Harassment, personal attacks, or discriminatory language.
- Publishing private information without consent.
- Repeated disruptive behavior in issues or pull requests.

## Enforcement
Project maintainers may edit or remove content, close discussions, or block contributors for violations.

## Reporting
Report conduct issues to the repository maintainer via private message or email.
